# m2clock - 24時間計アナログ時計 & AIアシスタントアプリ

m2clockは、1日の時間の流れを直感的に把握できる「24時間計」を採用した、ユニークでモダンなAndroid時計アプリです。

## 主な機能

### 1. 24時間計アナログ時計
- **24時間周期の文字盤**: 通常の12時間計ではなく、1日で1周する24時間計を採用。
- **正午が頂点**: 太陽が最も高い位置にある12時を一番上に配置し、生活リズムを視覚化。
- **動的な針の色**: 午前はオレンジ、午後はシステムのテーマカラーに短針の色が変化。
- **ゴールド三角形の秒針**: 文字盤の外周を回る、存在感のあるゴールドの秒針。

### 2. XEyes ギミック
- 時計の中心に配置された「目」が、外周を回る秒針を常に追いかけてキョロキョロと動きます。

### 3. 多機能情報表示
- **デジタル時刻**: 右上領域に大きく現在の時刻を表示。
- **日付・曜日**: 右下領域に日付と曜日を表示。
- **祝祭日対応**: 日本の祝祭日データを自動取得し、祝日と日曜日は赤色、土曜日は青色で表示。

### 4. カスタマイズ可能な時報機能
- 指定した時間帯（デフォルトは7時〜18時）に、正時（0分）になると通知音で時報を鳴らします。
- 右上の設定ボタン（ギヤマーク）から、時報のON/OFFや時間帯を簡単に調整可能。

### 5. レスポンシブデザイン
- **縦横自動切替**: スマホの向きに合わせてレイアウトが最適化されます。
- **ダークモード最適化**: 黒背景を基調とした、目に優しく視認性の高いデザイン。

### 6. AIアシスタント連携 (開発中)
- Google GenAI (Gemini) SDKを統合し、アプリ起動時に最新の情報を取得する仕組みを構築。

## 技術スタック
- **言語**: Kotlin
- **UIフレームワーク**: Jetpack Compose
- **非同期処理**: Coroutines, Flow
- **外部データ**: 
  - Holidays JP API (祝祭日情報取得)
  - Google GenAI SDK (Gemini API)

## 開発の目的
このアプリは、時間の流れをより深く意識し、1日を効率的かつ楽しく過ごすためのツールとして開発されました。
